# Create an App in Meta for Developers (Nuevo Sistema)

*(Documentación basada en: https://developers.facebook.com/docs/development/create-an-app/)*

Meta ha actualizado recientemente su flujo de creación de aplicaciones. En lugar de seleccionar un "Tipo de App" fijo desde el inicio (como Negocios, Juegos, etc.), ahora se configuran "Casos de Uso".

## Pasos para crear una aplicación:
1. Ir a **Mis Apps** en el panel de control.
2. Hacer clic en **Crear App**.
3. Aceptar el modal inicial del "nuevo sistema" haciendo clic nuevamente en **Crear app**.
4. Completar **Detalles de la app** (Nombre y Correo).
5. Seleccionar un **Caso de Uso** que se alinee con las necesidades de WhatsApp.
6. Especificar detalles del **Negocio** (Vincular un Administrador Comercial existente o crear uno nuevo). Esta parte es crítica para WhatsApp Business API.
7. Confirmar dependencias en **Requisitos**.

## Casos de Uso
Los casos de uso agrupan permisos y productos. Para WhatsApp, buscaremos el caso de uso relacionado con mensajería corporativa o integraciones de WhatsApp directamente si estuviera disponible, o continuaremos configurando la app para agregar el producto WhatsApp manualmente después.
