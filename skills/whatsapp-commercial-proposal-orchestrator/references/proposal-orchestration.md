# Proposal orchestration checklist

## Input pack
- Cliente y vertical
- Problema principal
- Estado actual de la operacion
- Objetivos de negocio
- Restricciones tecnicas y de presupuesto

## Sequence
1. Discovery
2. Packaging/Pricing
3. ROI
4. Implementation plan
5. Risk and governance
6. Proposal writing
7. Negotiation prep

## Done criteria
- Documento ejecutivo listo para enviar
- Documento detallado para validacion interna
- Script de llamada comercial de cierre
