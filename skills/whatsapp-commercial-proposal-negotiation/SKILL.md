# Negotiation and Closing Skill

## Overview

Aumentar tasa de cierre preparando respuestas para objeciones y una secuencia de cierre clara.

## Workflow

1. Detectar objeciones previsibles (precio, tiempo, integracion, soporte).
2. Preparar respuestas por valor y evidencia.
3. Definir concesiones permitidas y no permitidas.
4. Definir CTA de cierre (firma, anticipo, kickoff).
5. Preparar guion de reunion final.

## Output

- Objection matrix
- Guardrails de negociacion
- Script de cierre de 10 minutos
