# Implementation Plan Skill

## Overview

Definir roadmap realista con hitos semanales, responsabilidades y criterios de aceptacion.

## Workflow

1. Definir fases (Kickoff, Build, QA, Go-Live, Optimization).
2. Mapear entregables por fase.
3. Asignar responsables cliente/proveedor.
4. Definir dependencias criticas.
5. Definir plan de soporte post go-live.

## Suggested phase model

- Semana 1: Discovery final y aprobacion de alcance
- Semana 2-3: Configuracion flujos + dashboard
- Semana 4: QA y pruebas en entorno real
- Semana 5: Salida a produccion
- Semana 6+: Mejora continua y reportes

## Output

- Cronograma
- Matriz RACI simplificada
- Criterios go-live
