# Implementation checklist

## Before build
- Alcance firmado
- Accesos Meta y Vercel disponibles
- Responsables definidos

## Before go-live
- Flujo principal validado
- Escalamiento humano validado
- Registro de conversaciones validado
- Runbook compartido

## After go-live
- Monitoreo diario primera semana
- Reporte semanal de KPIs
- Backlog de mejoras priorizado
