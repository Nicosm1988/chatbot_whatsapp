# Proposal Writer Skill

## Overview

Convertir insumos tecnicos/comerciales en un documento convincente para decision ejecutiva.

## Writing structure

1. Resumen ejecutivo
2. Problema actual
3. Solucion propuesta
4. Alcance y entregables
5. Planes y precios
6. ROI esperado
7. Plan de implementacion
8. Riesgos y mitigaciones
9. Propuesta de proximo paso

## Style rules

- Claro, directo y sin jerga innecesaria
- Orientado a decision de negocio
- Con cifras y supuestos visibles
- Cierre con CTA concreto y fecha

## Output format

- Documento principal en markdown
- Version corta de una pagina para decision rapida
